    Class Ciphertext

    Provides helper methods for working with Ciphertext<THash>.

    Syntax
    public static class Ciphertext
    Remarks

    This type exists because of constraints with generics in .NET. Implementations of HashAlgorithm are conventionally created through a factory method like Create(). However, you can't call a static method on a type parameter.
