    Class Pipeline<T>

    Creates a function pipeline, calling each function or action on the value with postfix syntax.

    Remarks

    Pipeline<T> and Composer<TValue, TResult> differ in that Pipeline<T> invokes its functions eagerly, while Composer<TValue, TResult> invokes its functions lazily. Pipeline<T> produces a value as its final result, whereas Composer<TValue, TResult> produces a function.

    Examples

    Without Pipeline<T>:

    var result = Baz(Bar(Foo(value)));

    With Pipeline<T>:

    var result = Pipeline.Of(value)
    .Then(Foo)
    .Then(Bar)
    .Then(Baz)
    .Result;
